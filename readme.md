
# Monlabbing


## What does this project contain?

Monlabbing is a project complete with
- Frontend
    - Tailwind
    - CSS
    - HTML
    - Typescript
- Backend
    - Typescript
        - Express.js
    - Nginx
    - Postgresql
    - Postgresql replica
    - TimescaledDB
    - Redis (for flusing into TimescaledDB)
    - Redis (Hot cache)
    - Alerts with webhooks
    - Prometheus + Grafana
    - pqCryptography authentication
